# FocusFlow - Task Manager & Pomodoro

A beautiful Apple-inspired task manager with Pomodoro timer.

## Deploy to Vercel

### Option 1: Drag & Drop (Easiest)
1. Go to [vercel.com](https://vercel.com)
2. Sign in / Sign up
3. Drag the `index.html` file to the dashboard

### Option 2: GitHub
```bash
# 1. Create a GitHub repo
# 2. Push your code:
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/focusflow.git
git push -u origin main

# 3. Go to vercel.com
# 4. Import your repo
# 5. Deploy!
```

### Option 3: Vercel CLI
```bash
npm i -g vercel
vercel
```

## Features
- Task management with priorities
- Pomodoro timer (25/5/15 min)
- Dark mode
- LocalStorage persistence
- No database needed
